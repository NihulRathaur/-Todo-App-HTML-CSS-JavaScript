# Todo App

A simple and responsive Todo App built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Select due dates
- Delete tasks
- Responsive grid layout
- Clean and simple UI

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Project Preview

This project allows users to manage daily tasks easily with a minimal interface.

## How to Use

1. Enter a task in the input field
2. Select a due date
3. Click the "ADD" button
4. Delete tasks when completed

## Future Improvements

- Edit tasks
- Save tasks in Local Storage
- Dark mode
- Task completion status
